Stock Info & forecast destkop app - Java

In 8th grade I developed a java desktop app for displaying stock stock metrics and enabling forecasting. The user could insert different stock tickers getting all relevant metrics and expert evaluations. Moreover with using user input of market development expectations the app gave forecasts for probabale stock price developments. The app uses the yahoo finance API.
